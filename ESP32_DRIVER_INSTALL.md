# Установка драйверов для ESP32 с нативным USB

## Твоя ситуация
У тебя ESP32-S3 или ESP32-C3 с нативным USB (VID_303A&PID_1001). Windows видит устройство, но драйверы не установлены.

## Решение

### Способ 1: Через Диспетчер устройств (РЕКОМЕНДУЕТСЯ)

1. **Открой Диспетчер устройств** (Win+X → Диспетчер устройств)

2. **Найди устройство:**
   - В категории "Другие устройства" найди "USB JTAG/serial debug unit" (с желтым треугольником)
   - Или в "Порты" найди "Неизвестное устройство" на COM10

3. **Правый клик → Обновить драйвер**

4. **Выбери "Выполнить поиск драйверов на этом компьютере"**

5. **Выбери "Выбрать драйвер из списка уже установленных драйверов"**

6. **Нажми "Установить с диска"**

7. **Укажи путь к драйверам:**
   ```
   C:\Users\<ТВОЙ_ПОЛЬЗОВАТЕЛЬ>\.platformio\packages\tool-esptoolpy\esptool\targets\
   ```
   Или скачай драйверы отдельно (см. Способ 2)

### Способ 2: Скачать драйверы отдельно

1. **Скачай драйверы ESP32 USB:**
   - Официальный репозиторий: https://github.com/espressif/usb-serial-esp32
   - Или через ESP-IDF: https://docs.espressif.com/projects/esp-idf/en/latest/esp32s3/get-started/establish-serial-connection.html

2. **Установи драйверы:**
   - Распакуй архив
   - Запусти установщик от имени администратора
   - Или установи вручную через Диспетчер устройств (указав путь к папке с драйверами)

### Способ 3: Использовать CH340 адаптер (если есть)

Если у тебя есть ESP32 с CH340 адаптером (не нативный USB):
1. Установи драйверы CH340: http://www.wch.cn/downloads/CH341SER_EXE.html
2. Используй этот порт вместо нативного USB

## После установки

1. **Перезагрузи компьютер**
2. **Отключи и подключи ESP32 заново**
3. **Проверь:**
   ```powershell
   pio device list
   ```
4. **Должен появиться COM-порт** (например, COM10 или другой)

## Если не помогло

1. Попробуй другой USB порт
2. Попробуй другой USB кабель (не все кабели поддерживают передачу данных)
3. Проверь, что плата поддерживает USB (не все ESP32 имеют нативный USB)
4. Попробуй установить ESP-IDF - он включает драйверы: https://docs.espressif.com/projects/esp-idf/en/latest/esp32s3/get-started/

## Альтернатива: Использовать CH340 порты

У тебя есть CH340 адаптеры на COM3/4/5/6/11. Если установишь драйверы CH340, можешь использовать их:
1. Установи CH340 драйверы: http://www.wch.cn/downloads/CH341SER_EXE.html
2. В `platformio.ini` укажи конкретный порт:
   ```ini
   upload_port = COM4
   monitor_port = COM4
   ```
