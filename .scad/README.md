# 3D Модель корпуса для ESP32 + OLED + Aqara розетка

## Компоненты модели

Модель состоит из 3 частей:
1. **main_case** - основной корпус (по умолчанию активен)
2. **lid** - крышка с отверстием для OLED (закомментирован)
3. **aqara_adapter** - адаптер для крепления к Aqara розетке (закомментирован)

## Размеры компонентов

- **ESP32 DevKit V1**: 25.4mm × 53.3mm × 13mm
- **OLED SSD1306**: 27mm × 27mm × 4mm
- **Aqara накладная розетка**: ~85mm диаметр

## Размеры корпуса

- **Внутренние**: ~57mm × ~60mm × ~17mm
- **Внешние**: ~61mm × ~64mm × ~19mm
- **Толщина стенок**: 2mm

## Печать

### Настройки принтера:
- **Слой**: 0.2mm
- **Заполнение**: 20-30%
- **Поддержки**: не требуются
- **Ориентация**: дном вниз

### Печать частей:

1. **Основной корпус** (main_case):
   - Время печати: ~1-2 часа
   - Материал: PLA/PETG
   - Ориентация: дном вниз

2. **Крышка** (lid):
   - Раскомментируй строки 167-168 в cover.scad
   - Время печати: ~30 минут
   - Ориентация: плоской стороной вниз

3. **Адаптер Aqara** (aqara_adapter):
   - Раскомментируй строки 171-172 в cover.scad
   - Время печати: ~1 час
   - Ориентация: дном вниз

## Сборка

1. Установи ESP32 в корпус (USB разъем в отверстие)
2. Установи OLED модуль (дисплей должен быть виден через отверстие в крышке)
3. Подключи провода I2C (SDA, SCL) к ESP32
4. Закрепи крышку винтами M2.5
5. При необходимости установи адаптер на Aqara розетку

## Крепление

- Используй винты M2.5 × 8-10mm для крепления крышки
- Адаптер Aqara надевается на розетку и фиксирует корпус

## Настройка размеров

Если твои компоненты отличаются по размерам, измени переменные в начале файла:
- `esp32_w`, `esp32_l`, `esp32_h` - размеры ESP32
- `oled_w`, `oled_l`, `oled_h` - размеры OLED
- `aqara_diameter` - диаметр Aqara розетки

## Примечания

- Отверстие для USB разъема может потребовать подгонки под твою модель ESP32
- Размер отверстия для OLED можно скорректировать в модуле `lid()`
- Вентиляционные отверстия опциональны, можно убрать
