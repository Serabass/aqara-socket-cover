# Установка драйверов для ESP32

## Проблема
Windows видит USB-to-Serial адаптеры (CH340/CH9102), но они помечены как "Unknown Ports" - драйверы не установлены.

## Решение

### Вариант 1: Автоматическая установка (рекомендуется)

1. Открой **Диспетчер устройств** (Win+X → Диспетчер устройств)
2. Найди устройства с желтым треугольником:
   - "USB-SERIAL CH340" на COM4, COM5, COM6, COM11
   - "USB-Enhanced-SERIAL CH9102" на COM3
3. Правый клик → **Обновить драйвер**
4. Выбери **Автоматический поиск драйверов**
5. Windows попробует найти драйверы через Windows Update

### Вариант 2: Ручная установка драйверов

1. Скачай драйверы CH340:
   - Официальный сайт: http://www.wch.cn/downloads/CH341SER_EXE.html
   - Или: https://github.com/nodemcu/nodemcu-devkit/tree/master/Drivers

2. Скачай драйверы CH9102:
   - Официальный сайт: http://www.wch.cn/downloads/CH9102DRV_EXE.html

3. Установи драйверы (запусти .exe файлы)

4. Перезагрузи компьютер или отключи/подключи ESP32 заново

### Вариант 3: Использование готовых драйверов через PowerShell

Запусти от имени администратора:
```powershell
# Скачать и установить драйверы CH340
# (нужно будет скачать вручную и указать путь)
```

## Проверка после установки

После установки драйверов проверь:
```powershell
pio device list
```

Должны появиться COM-порты (COM3, COM4, COM5, COM6, COM11).

## Если не помогло

1. Попробуй другой USB порт
2. Попробуй другой USB кабель (не все кабели поддерживают передачу данных)
3. Проверь, что плата ESP32 поддерживает USB (не все модели имеют встроенный USB-to-Serial)
